# Quirky Help — Microsoft Store Listing Draft

## App name
Quirky Help

## Short description
A Windows troubleshooting helper that explains screenshots, terminal errors, and setup commands in plain English.

## One-line pitch
Get clearer explanations and safer next steps for Windows errors, setup issues, and command-line problems.

## Long description
Quirky Help is a Windows desktop app that helps you understand confusing setup problems, terminal errors, and app screens.

Paste a screenshot or describe the issue, and Quirky Help organizes the answer into practical guidance:

- what happened
- where to run the fix
- what the command does
- what to do next
- what result to expect
- what to try if it fails

Quirky Help is designed to make troubleshooting easier without taking control of your computer. It does not auto-run commands or make system changes. It explains the issue, suggests a next step, and lets you decide what to copy or run.

Use Quirky Help for:

- Windows setup and installation issues
- PowerShell or CMD error messages
- developer tool setup problems
- screenshots that need a plain-English explanation
- command suggestions with safety context

Quirky Help supports Gemini, Groq, OpenAI, and custom OpenAI-compatible providers. You bring your own API key, choose your provider during setup, and can update settings at any time.

Your provider settings are stored locally on your device. Prompts and screenshots are sent only to the AI provider you configure.

## Key features
- Explain screenshots and terminal errors in plain English
- Get Windows-friendly command guidance
- See where to run a command before copying it
- Understand what a command does before using it
- View expected results and fallback steps
- Copy PowerShell or CMD suggestions manually
- Set up Gemini, Groq, OpenAI, or a custom compatible provider
- Review built-in privacy and safety information
- Keep provider settings stored locally on your device

## Who it is for
- Windows users troubleshooting setup problems
- people learning PowerShell or command-line basics
- developers and makers debugging install issues
- anyone who wants clearer guidance from screenshots or error messages

## Trust and privacy
Quirky Help does not auto-run commands or make changes to your computer. Commands are suggestions only and should be reviewed before use.

API keys and provider settings are stored locally on your device. Prompts and screenshots are sent only to the AI provider you select in the app settings.

Screenshots may contain sensitive information. Review screenshots before sending them to your configured AI provider.
