# Quirky Help — Publish Privacy Policy

## Goal
Get `privacy-policy.html` onto a stable public URL that can be used in the Microsoft Store submission.

## Easiest publish options

### Option 1 — GitHub Pages (recommended)
Good if Katie wants the fastest low-maintenance path.

Suggested URL shape:
- `https://m-lwatcher.github.io/quirky-help/privacy-policy.html`
- or any equivalent GitHub Pages path

Steps:
1. Put `privacy-policy.html` in a repo/folder that can be published with GitHub Pages.
2. Enable GitHub Pages for that repo.
3. Confirm the page loads publicly.
4. Paste the final public URL into:
   - `STORE_LISTING_DRAFT.md`
   - `MICROSOFT_STORE_FINAL_COPY.md`
   - `FINAL_SUBMISSION_TODO.md`

### Option 2 — Existing personal/site host
Good if Katie already has a simple domain or static host.

Suggested URL shape:
- `https://<domain>/quirky-help/privacy-policy`
- or `https://<domain>/privacy/quirky-help`

### Option 3 — Temporary static host
Use any stable static host if GitHub Pages is annoying.

## Recommendation
Use **GitHub Pages** first unless Katie already has a better place.
