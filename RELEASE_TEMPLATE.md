## Title
Quirky Help vX.Y.Z

## Summary
Quirky Help is a Windows desktop troubleshooting helper for screenshots, terminal errors, and setup commands.

## What's in this release
- improved Windows troubleshooting flow
- screenshot support
- clearer command guidance
- local provider setup for Gemini, Groq, OpenAI, and compatible providers
- About / Privacy view

## Important notes
- Quirky Help does **not** auto-run commands.
- Commands are suggestions only.
- You bring your own AI API key.
- Prompts and screenshots are sent only to the provider you configure.

## Install
1. Download the Windows installer from this release.
2. Run the installer.
3. Launch **Quirky Help**.
4. Complete provider setup on first run.
